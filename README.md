# multi-agent-who-is-undercover
基于LLM multi agents的《谁是卧底》游戏模拟。


## 安装
```pip install -r requirements.txt```

## 修改.env

```
AWS_ACCESS_KEY_ID="xxxxx"
AWS_SECRET_ACCESS_KEY="xxxxx"
AWS_REGION="xxxxxx"
```

## 运行
```streamlit run who_is_undercover_frontend.py```
